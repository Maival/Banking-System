Banking Application (Java)

Description :

This is a console-based Banking Application built using Java. It allows users to perform basic banking operations like viewing balance, depositing money, and withdrawing funds through a simple menu-driven interface.

Features : 

- View current account balance
- Deposit money with validation
- Withdraw money with balance checking
- Input validation using exception handling
- User-friendly menu-driven interface
- Option to continue or exit after each operation
